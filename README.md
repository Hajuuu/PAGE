# 독서 기록 공유 플랫폼, PAGE
## 프로젝트 소개


사용자가 읽고 있는 도서를 등록하고 독서 감상을 작성•공유할 수 있는 독서 기록 공유 플랫폼

## 사용 기술


### Frontend

- Thymeleaf – 서버 사이드 템플릿 엔진


### Backend

- Spring Boot
    - Spring MVC
    - Spring Security
    - Spring Data JPA

### Database / Storage

- MySQL (JPA, QueryDSL 기반 데이터 처리)


### Language

- Java 17

## 프로젝트 정보


기간: ~2025.11

**개인 프로젝트**

- 프로젝트 기획
- 전체 시스템 아키텍처 설계
- 백엔드·프론트엔드 개발 전반 담당

## 프로젝트 상세


### 주요 개발 기능

**도서 검색 및 등록 기능**

- 네이버 도서 API, 알라딘 도서 API를 이용한 도서 정보 검색
- 읽은 도서를 등록하고 관리할 수 있는 기능

**독서 기록 저장 기능**

- 현재 독서 상황을 관리하고 독서 기록을 남길 수 있다.

**독서 기록 공유 기능**

- 독서 기록을 다른 이용자와 공유할 수 있는 기능
- 팔로우 및 팔로잉을 통해 구현
